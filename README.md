**Electric Vehicle Market Analysis Dashboard (Power BI)**

**Overview**

This project analyzes electric vehicle adoption trends across the United States using real-world population data. The goal is to uncover insights into growth patterns, manufacturer dominance, and regional distribution.

**Key Insights**

- EV adoption significantly increased after 2020
- Tesla dominates the market with ~68K vehicles
- BEVs account for ~78% of total EVs
- Certain states show higher adoption rates, indicating regional demand differences
- Tools Used
- Power BI (DAX, Power Query)
- Excel (data cleaning)
- Dataset

[Electric Vehicle Population dataset (public dataset)](https://www.kaggle.com/datasets/utkarshx27/electric-vehicle-population-data?select=Electric_Vehicle_Population_Data.csv)
